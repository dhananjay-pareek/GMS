Configuration Guide

Search a place (Map) Type a location name (e.g., "Seattle, WA") into the map search bar and click "Find". The map will automatically fetch the coordinates for you.
Job Name An optional friendly name to identify your scraping job in the results table.
Keywords The search terms you want to scrape (e.g., "restaurants in New York"). Put one search per line.
Language The 2-letter language code for the Google Maps results (e.g., "en" for English, "es" for Spanish).
Location Settings (Map / Coordinates) Set a specific central location using the interactive map or by entering exact Latitude and Longitude. Used with Radius to define a search area.
📍 My Location Automatically detects and sets your current geographical coordinates on the interactive map.
Clear Removes the selected location from the map and resets coordinates.
Zoom Level The zoom level of the map. Higher zoom means a closer view of the location.
Fast Mode (BETA) Speeds up scraping by using a grid-coordinate technique instead of mimicking user scrolling. Requires exact Latitude and Longitude.
Suggestion: Leave ON for faster execution. Provide coordinate targets.
Radius The search radius in meters around the central coordinate point (e.g. 5000 for 5km).
Criteria: Numbers only, 0 to disable.
Suggested Value: 5000 to 10000.
Depth How far deep to scrape the results (number of scrolls or pages). Higher means more results but slower execution.
Criteria: Integer > 0.
Suggested Value: 120 (for a thorough deep scrape).
Fetch Emails When enabled, the scraper visits business websites to attempt to extract email addresses and social links.
Suggestion: Turn ON if outreach metrics are critical.
Max Job Time The maximum allowed running time before the job forcefully stops safely. Very important to prevent stuck jobs.
Criteria: Time format like Xm or Xh (e.g., 30m, 2h).
Suggested Value: 1h or 2h.
Proxies A list of HTTP proxies to use, one per line (IP:PORT or user:pass@IP:PORT). Helpful to avoid Google rate limit blocks.
Suggestion: Use a proxy rotation pool if scraping large cities.
Start Scraping Submits the configuration and queues the scraping job.
